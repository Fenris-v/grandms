Первая мебельная фабрика

Предприятие с богатой историей, расположенное в Санкт-Петербурге.

Сегодня предприятие располагает самым современным производством корпусной мебели в России и поставляет на российский рынок и рынки других стран мебель для кухни, шкафы, прихожие, офисную мебель, мебель для гостиниц, ресторанов и других общественных учреждений.

Компания предлагает различные стилевые решения комфортабельной и функциональной мебели - от классики до хай-тека.

Какие задачи стояли перед командой:

  • Увеличение количества обращений от B2C сегмента и снижение стоимости такого обращения.
  • Переход к понятной для клиента форме отчетности с прозрачными показателями.
  • Рост присутствия в регионах.
  • Доработка юзабилити сайта.
Контекстная реклама
  • Значительное расширение семантического пула, проработка всех ключевых направлений запросов - транзакционные, стилевые, спецификационные, ценовые, акционные, конкурентные, брендовые и др.
  • Выстраивание отдельных стратегий для разных направлений в контексте за счет корректировок и детальной настройки кампаний
  • Настройка большого количества ретаргетинговых и ремаркетинговых кампаний для разных аудиторий с целью наиболее эффективного взаимодействия с ними
  • Использование гео-сегментов по локациям салонов для увеличения трафика в конкретных точках продаж
  • Разработка специальной механики и тайминга акционных предложений
  • Активное использование Яндекс.Экспериментов для проверки различных гипотез по клиенту
  • Отстройка всех сторон аналитики, подключение системы колл-трекинга,изменение структур целей
Медийная реклама
  • Использование большого количества медийных форматов с различными креативами для роста охвата площадок и аудиторий
  • Запуск комплексных рекламных кампаний для поддержки рекламы на тв - рк в youtube и кмс с тестированием различных таргетингов по намерениям и ремаркетинговым аудиториям
  • Использование возможностей контекстной и медийной рекламы для быстрого вывода на рынок
  • Услуги по полной меблировке квартир и моментального получения лидов по ней для отдела продаж клиента
SEO-продвижение и веб-разработка
  • Создание структуры региональных поддоменов для ранжирования сайта по 26 регионам России (изначально - 1 регион)
  • Проработка посадочных страниц под каждое предложение клиента для увеличения их привлекательности и вовлеченности пользователя => увеличение поведенческих показателей
  • Значительное увеличение количества уникальных текстов под посадочные
  • Расширение структуры сайта, чтобы представить потенциальному покупателю полностью релевантную страницу
  • Разработка уникальных лендингов под дополнительные услуги клиента со своим дизайном и формами
  • Работа с юзабилити сайта для увеличения конверсионности и удобства для покупателя
Дополнительные рекламные инструменты

  • Получение обратной связи по привлекательности новых кухонных моделей с помощью инструмента взгляд

  • С помощью гео-рекламы в яндексе и размещения через программатик-платформы - увеличение узнаваемости бренда и дополнительный рост заявок

  • Использование инструмента яндекс.коллекции для увеличения присутствия в поисковой выдаче по ключевым запросам
